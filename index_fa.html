مفسر Q+ 🧮

به مفسر Q+ خوش آمدید، یک ماشین حساب قدرتمند مبتنی بر پایتون که از نشانه‌گذاری پست‌فیکس (نشانه‌گذاری لهستانی معکوس) برای انجام عملیات مختلف ریاضی استفاده می‌کند. 🚀

ویژگی‌ها 🌟

  • عملیات ریاضی پایه
  • توابع ریاضی پیشرفته
  • محاسبات مثلثاتی
  • ذخیره‌سازی و استفاده از متغیرها
  • پیاده‌سازی توابع سفارشی

چگونه کار می‌کند 🔍

مفسر Q+ از یک پشته برای ارزیابی عبارات پست‌فیکس استفاده می‌کند، پشتیبانی از:

  • افزودن اعداد به پشته
  • اجرای عملیات
  • اختصاص متغیرها
  • ارزیابی عبارات

مثال استفاده 💻

> 3 4 +
پشته بعد از '3': [3.0]
پشته بعد از '4': [3.0, 4.0]
پشته بعد از '+': [7.0]
نتیجه: 7.0

> x 5 =
متغیر x مقدار: 5.0 اختصاص داده شد

> x 3 *
پشته بعد از 'x': [5.0]
پشته بعد از '3': [5.0, 3.0]
پشته بعد از '*': [15.0]
نتیجه: 15.0

> 90 sin
پشته بعد از '90': [90.0]
پشته بعد از 'sin': [1.0]
نتیجه: 1.0
