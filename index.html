How It Works üîç

Q+ Interpreter uses a stack to evaluate postfix expressions, supporting:

  • Number pushing
  • Operation performing
  • Variable assignment
  • Expression evaluation

Usage Example üíª

> 3 4 +
Stack after '3': [3.0]
Stack after '4': [3.0, 4.0]
Stack after '+': [7.0]
Result: 7.0

> x 5 =
Variable x assigned value: 5.0

> x 3 *
Stack after 'x': [5.0]
Stack after '3': [5.0, 3.0]
Stack after '*': [15.0]
Result: 15.0

> 90 sin
Stack after '90': [90.0]
Stack after 'sin': [1.0]
Result: 1.0
